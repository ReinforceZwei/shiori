# Shiori Chan Browser Extension

Quickly add bookmark to Shiori Chan instance. Support getting website metadata.

## Extension Feature

- Shortcut key to add current tab to bookmark (like browser built-in bookmark)
- Get website metadata (title, icon) from browser tab context
- Integrate with collection
- Support self-hosted Shiori Chan instance
- Authenticate with Shiori Chan using API key

## Permission

- Tab: get `title`, `url` and `favIconUrl`
- Storage: save extension config (self-hosted instance info, API key)
